
  # Reading Comprehension App

  This is a code bundle for Reading Comprehension App. The original project is available at https://www.figma.com/design/KIRnobZJKIG7X1QQS8LVAV/Reading-Comprehension-App.

  ## Setup

  ### Environment Variables

  Create a `.env` file in the root directory with the following variable:

  ```
  VITE_GEMINI_API_KEY=your_api_key_here
  ```

  Get your Gemini API key from: https://makersuite.google.com/app/apikey

  **Note:** The `.env` file is already in `.gitignore` and will not be committed to the repository.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
